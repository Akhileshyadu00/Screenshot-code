# Screenshot to Code SaaS

A premium, responsive React application that converts UI screenshots into production-ready HTML + Tailwind CSS code using OpenAI's GPT-4o Vision model.

## Features

- **Drag & Drop Upload**: Smooth file upload experience.
- **Real AI Conversion**: Uses your OpenAI API Key to generate code.
- **Live Preview**: secure iframe rendering of generated code.
- **Code Editor**: View and copy generated code with syntax highlighting.
- **Responsive Design**: Mobile-first UI with a polished dark mode.
- **Privacy Focused**: API Key is stored locally in your browser.

## Tech Stack

- Vite + React
- Tailwind CSS
- OpenAI SDK (Browser-compatible)
- Lucide React (Icons)
- React Syntax Highlighter

## Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Akhileshyadu00/Screenshot-code.git
    cd Screenshot-code
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run locally**
    ```bash
    npm run dev
    ```

4.  **Use the App**
    - Open `http://localhost:5173`
    - Enter your OpenAI API Key in the Sidebar (Settings > Advanced)
    - Upload a screenshot to generate code!
